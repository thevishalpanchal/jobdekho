# Ardhnarishwar

AI Powered Job Portal using React, Spring Boot and MySQL.

## Frontend Setup

```bash
npm install
npm start
```

Frontend runs on:

```text
http://localhost:3000
```

## Backend Setup

```bash
mvn clean install
mvn spring-boot:run
```

Backend runs on:

```text
http://localhost:8085
```

## Technologies Used

* React JS
* Spring Boot
* MySQL
* JWT Authentication
* Axios

## Project Structure

```text
frontend/
backend/
```

## Notes

* Frontend API base URL is configured in:

```text
src/config.js
```

* Change API URL during deployment.
